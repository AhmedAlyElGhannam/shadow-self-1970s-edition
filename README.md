# Shadow-Self: A Chip-8 Emulator

## Introduction
> A simple chip-8 emulator written in C. Why? Because I wanted to get a better idea on how hardware was emulated so I decided to do it myself on a small scale. I created and ran it on Linux + utilized [SDL](https://www.libsdl.org/) library to make my life easier when drawing pixels/playing beeps.

## Resources
> The main resource I followed was [Cowgod's Chip-8 Technical Reference](http://devernay.free.fr/hacks/chip8/C8TECH10.HTM). The structure of the project itself is heavily inspired by [Daniel McCarthy's own Chip-8 written in C](https://github.com/nibblebits); huge shout out to him.
